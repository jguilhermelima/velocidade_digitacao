<h1 align="center"> Velocidade Digitação </h1>

<h2>Descrição do Projeto</h2>
Este código tem como funcionalidade testar a velocidade de digitação.
O usuário tem um minuto para digitar a frase que aparece na tela. Ao fim do tempo, são exibidos o WPM (palavras por minuto), que mede a velocidade da digitação, e a sua precisão ao escrever.


Passo a passo:

* Clicar no botão iniciar.
  
* Digitar a frase que aparece na tela.

<h2>🛠️ Construído com</h2>

* Python

* Tkinter
